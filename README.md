# assembler
I wrote a single-pass MIPS assembler.


It supports the following MIPS instructions:

  - `add`

  - `addi`

  - `sub`

  - `and`

  - `andi`

  - `or`

  - `ori`

  - `xor`

  - `xori`

  - `nor`

  - `sll`

  - `sra`

  - `srl`

  - `slt`

  - `slti`

  - `beq`

  - `bne`

  - `j`

  - `jal`

  - `jr`

  - `lw`

  - `sw`

  - `nop`
